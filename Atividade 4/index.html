1. Introdução

A Especificação de Requisitos de Software (SRS) define os requisitos detalhados para o desenvolvimento do Sistema de Gestão de Clínicas Veterinárias da empresa Balti. O objetivo deste documento é fornecer uma visão abrangente das funcionalidades, características e restrições do sistema a ser desenvolvido.

2. Objetivos

O Sistema de Gestão de Clínicas Veterinárias tem como objetivo principal otimizar e automatizar processos relacionados à gestão de clínicas veterinárias. Ele visa oferecer soluções eficientes para o agendamento de consultas, registro de pacientes e seus proprietários, gestão de procedimentos médicos, controle de estoque e faturamento.
