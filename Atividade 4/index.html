1. Introdução

A Especificação de Requisitos de Software (SRS) define os requisitos detalhados para o desenvolvimento do Sistema de Gestão de Clínicas Veterinárias da empresa Balti. Este documento proporciona uma visão abrangente das funcionalidades, características e restrições do sistema proposto.

2. Objetivos

O Sistema de Gestão de Clínicas Veterinárias tem como objetivo aprimorar a eficiência e automatizar os processos relacionados à gestão de clínicas veterinárias. Ele visa oferecer soluções eficazes para o agendamento de consultas, registro de pacientes e proprietários, gestão de procedimentos médicos, controle de estoque e faturamento.

3. Escopo do Projeto

O sistema abrangerá as seguintes funcionalidades principais:

  • Cadastro de Pacientes e Proprietários
  • Agendamento de Consultas e Procedimentos
  • Registro de Diagnósticos e Tratamentos
  • Controle de Estoque de Medicamentos e Produtos
  • Faturamento e Emissão de Notas Fiscais

4. Requisitos Funcionais

4.1 Cadastro de Pacientes e Proprietários

O sistema permitirá o cadastro de informações detalhadas sobre pacientes e seus proprietários, incluindo nome, espécie, raça, data de nascimento, histórico médico e informações de contato.
